Created by Google Engrs. in 2007

Because they dislike C++

Actually, they created it to improve their productivity in an era of multicore, networked machines and large codebases. The designers wanted to address criticism of other languages in use at Google, but keep their useful characteristics: Static typing and run-time efficiency (like C)

The initial compiler was written in C, until they're able to create their own compiler written in Go itself.

What's cool about Go?

  • Goroutines and Channeling
  • Its easy to code, maintain, writing structs is like writing object instance, as if the single source of truth.
  • Either a functional or OOP dev can easily understand the code
  • Test cases' runs in parallel, you can write test cases along with the go lang file similar in node.
  • You can create your own gRPC
  • You can write your own WebAssembly

How fast can you learn Go?

  • Depends, at least 2 weeks ~ 4 weeks
  • The most effective thing to do is to compare syntax from your known language agains't Go language, such as if/switch/for/errors/returns/data types

Struct

							
								package main

								type User struct {
									FirstName string
									LastName string
									Age int8
									BirthYear interface{}
								}

								func NewUser(fName string, lName string, age int8) *User {
									return &User{
										FirstName: fName,
										LastName: lName,
										Age: age,
									}
								}

Under the main method, we create a user from User{}

							
								// ...
								func main() {
									user := NewUser("Daison", "Cariño", 31)
									fmt.Println(user.FirstName, user.LastName, user.Age)
								}
								> Output: Daison Cariño 31

How about using those struct's data as a method

							
								// ...
								func (user User) getFullName() string {
									return fmt.Sprintf("%s %s", user.FirstName, user.LastName)
								}

								func (user *User) getBirthYear() int8 {
									if user.BirthYear != nil {
										return user.BirthYear.(int8)
									}
									year, _, _ := time.Now().Date()
									user.BirthYear = year - user.Age
									return user.BirthYear
								}

Now go back to main method

							
								// ...
								func main() {
									user := NewUser("Daison", "Cariño", 31)
									fmt.Println(user.getFullName(), user.Age, user.getBirthYear())
								}
								> Output: Daison Cariño 31 1991
